# NawthViper Currency Bot 💰

An automated currency trading bot built with Python and Node.js.

## 🚀 Features
- Real-time data analysis
- Smart risk management
- Multi-timeframe structure detection
- Frontend dashboard for monitoring trades

## 🧠 Tech Stack
- **Backend:** Python (FastAPI / Flask)
- **Frontend:** React / Tailwind
- **Database:** PostgreSQL
- **Broker Integration:** MT5 API

## ⚙️ Overview

**NawthViper Currency Bot** is a powerful automated trading system designed for high-precision forex and synthetic index trading.  
It combines **Smart Money Concepts (SMC)**, **risk management**, and **multi-timeframe structure analysis** to deliver intelligent trading decisions.

---

## 🚀 Features

✅ Automated order execution on MT5  
✅ Smart Money Concepts structure detection (CHoCH, BOS, OB, etc.)  
✅ Real-time trade monitoring dashboard (frontend)  
✅ Integrated risk and balance tracking  
✅ Optimized backend for scalability and stability
